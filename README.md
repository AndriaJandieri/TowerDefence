# Tower Defense (Unity WebGL Demo)

A simple Tower Defense demo built in Unity using ScriptableObjects for towers, enemies, and waves.  
This demo contains one level showcasing the core gameplay systems.

## Play the Demo
[Play on Itch.io](https://andriajandieri.itch.io/towerdefence)

## Features
- Modular towers and enemies using ScriptableObjects  
- Configurable wave system  
- Basic enemy pathing and tower targeting  
- Projectile system  
- One-level demo
- WebGL optimized build
